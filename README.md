# Job Management Admin Interface

Full-stack application for managing job postings with Next.js, Mantine, NestJS, and MongoDB.

## Setup

### Backend
```bash
cd backend
npm install
npm run start:dev
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

## Features
- Job listing with filters
- Create job postings
- MongoDB integration
- Responsive UI matching Figma design
